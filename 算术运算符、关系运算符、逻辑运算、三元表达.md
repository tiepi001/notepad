## 关系运算

> >,<,>=,<=
关系运行结果是布尔值,判断是否

关系运算符会自动对两边类型进行转换


> 例:‘22’>12; true


==:只判断内容,不判断类型

===:全等,即判断内容,也判断类型

!=:只判断内容,不判断类型

!==:即判断内容,也判断类型



## 逻辑运算


！(非):
&(与):只要有一边为fale, 那么就是false
|(或):要有一边为true, 那么就是true
^(异或):只要是相同的boolean值, 那么就是false, 不相同才是true

- 解释: 用来连接boolean(关系表达式)表达式,或者是boolean取值都可以, 表达式返回的结果还是一个boolean类型

&&(双与):一假则假，全真才为真

||(双或):一真则真，全假才为假

- 解释: 双与和双或 得到的结果 和 单与和单或 是完全一样的!

区别: 
 

### && 和 & :
 
- && 一旦判断到左边是false之后, 右边就不参与运算了
 
- &: 不管左边是true还是false, 右边都参与运算
 

### || 和 |
 

- || 一旦判断左边是true之后, 右边就不参与运算了
 
- |  不管左边是true还是false, 右边都参与运算
 
**注意: 对于只是要判断一个表达式的结果来讲, 双与, 双或的计算效率高些!**




## 三元表达


> 
var 变量=条件？条件为真的值:条件为假的值
```
  int a=1;
  int b=2;
  int c= a>b?1:0
  //条件不成立  c的值为0
```

![图片](https://gss0.baidu.com/9vo3dSag_xI4khGko9WTAnF6hhy/zhidao/wh%3D600%2C800/sign=10b7e6f26c09c93d07a706f1af0dd4e4/8c1001e93901213f61aee2205fe736d12e2e95a9.jpg)

三元如名字表示的三元运算符需要三个操作数。

语法是 条件 ? 结果1 : 结果2;. 


这里你把条件写在问号(?)的前面后面跟着用冒号(:)分隔的结果1和结果2。满足条件时结果1否则结果2。


**拓展：** if...else... 和 三元表达式
```
if(a || b){
  不挂科;
}else{
   门门挂;
}
```
那么三元的表示法就是

代码如下:
```
a || b ? 不挂科 : 门门挂
```
